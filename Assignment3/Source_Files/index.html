This is a web application that is hosted on Compute engine using Nginx, Flask, and Gunicorn. The application does Sentiment Analysis using Google Cloud AI service named as Natural Language API on user provided data and then stores that data and its analysis on Cluod Mysql server and then shows the result to the user.
The user can also see the complete database.
